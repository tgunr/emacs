3.7.13 Arithmetic Tutorial Exercise 2

27 <RET> 9 B could give the exact result ‘3:2’, but it does a floating-point calculation instead and produces ‘1.5’.

Calc will find an exact result for a logarithm if the result is an integer or (when in Fraction mode) the reciprocal of an integer. But there is no efficient way to search the space of all possible rational numbers for an exact answer, so Calc doesn't try.
