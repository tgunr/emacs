8 Credits

Ediff was written by Michael Kifer <kifer@cs.stonybrook.edu>. It was inspired by emerge.el written by Dale R. Worley <drw@math.mit.edu>. An idea due to Boris Goldowsky <boris@cs.rochester.edu> made it possible to highlight fine differences in Ediff buffers. Alastair Burt <burt@dfki.uni-kl.de> ported Ediff to XEmacs, Eric Freudenthal <freudent@jan.ultra.nyu.edu> made it work with VC, Marc Paquette <marcpa@cam.org> wrote the toolbar support package for Ediff, and Hrvoje Niksic <hniksic@xemacs.org> adapted it to the Emacs customization package.

Many people provided help with bug reports, feature suggestions, and advice. Without them, Ediff would not be nearly as useful as it is today. Here is a hopefully full list of contributors:
