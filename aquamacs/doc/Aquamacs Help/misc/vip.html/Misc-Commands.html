1.4.13 Miscellaneous Commands

X (
C-x (

Start remembering keyboard macro.

X )
C-x )

Finish remembering keyboard macro.

*

Call last remembered keyboard macro.

X Z
C-x C-z

Suspend Emacs.

Z Z

Exit Emacs.

Q

Query replace.

R

Replace.
