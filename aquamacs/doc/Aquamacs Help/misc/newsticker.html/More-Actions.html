5.4 More Actions

Open the link to the full article (as contained in the current headline) in your web browser newsticker-treeview-browse-url).

Get News

You can force immediate download of news with the following commands.

g

Get news for currently shown feed (newsticker-treeview-get-news).

G

Get news for all feeds (newsticker-get-all-news).

Add More Feeds

a

The command newsticker-add-url prompts for an URL and a name of a new feed. It then prepares a customization buffer where the details of the new feed can be set.
