New features in Ma Gnus:

  • Changes in summary and article mode
    • By default, MIME part buttons for attachments (if any) will appear in the end of the article header in addition to the bottom of the article body, so you can easily find them without scrolling the article again and again. See MIME Commands.
  • Changes in Message mode and related Gnus features
    • The new hooks gnus-gcc-pre-body-encode-hook and gnus-gcc-post-body-encode-hook are run before/after encoding the message body of the Gcc copy of a sent message. See See Archived Messages.
