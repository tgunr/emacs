14 AUCTeX

AUCTeX is without doubt the best major mode for editing TeX and LaTeX files with Emacs (see AUCTeX in The AUCTeX User Manual). If AUCTeX is not part of your Emacs distribution, you can get it9 by FTP from the AUCTeX web site.
