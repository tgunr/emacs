7.3 Enabling EDT Control Key Sequence Bindings

Where EDT key bindings and Emacs key bindings conflict, the default Emacs key bindings are retained by default. Some diehard EDT users may not like this. So, if the variable edt-use-EDT-control-key-bindings is set to true in a user’s .emacs file, then the default EDT Emulation mode will enable most of the original EDT control key sequence bindings. If you wish to do this, add the following line to your .emacs file:

(setq edt-use-EDT-control-key-bindings t)
